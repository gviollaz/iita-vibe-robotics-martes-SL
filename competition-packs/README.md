# 🏆 Competition Packs

Cada carpeta contiene información específica de una competencia:
reglas, specs, scoring, y notas del equipo.

## Competencias activas del IITA

| Competencia | Carpeta | Estado |
|-------------|---------|--------|
| RoboCup Junior Soccer | `rcj-soccer/` | Equipo IITA Salta (campeones nacionales 2025) |
| FLL Challenge | `fll/` | Para referencia y práctica |
| WRO RoboMission | `wro-robomission/` | Para referencia y práctica |

## Cómo usar

1. Antes de empezar a programar para una competencia, leé su `spec.md`
2. Usá el skill `rulebook-to-spec` para generar el spec a partir de las reglas oficiales
3. Guardá las reglas oficiales como PDF en la carpeta correspondiente
4. Actualizá el spec si hay Q&A o updates oficiales
