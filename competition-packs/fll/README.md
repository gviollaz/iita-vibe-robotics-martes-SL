# 🤖 FIRST LEGO League Challenge

## Referencia para el taller

### Resumen

- Robot game: 2:30 minutos por ronda
- Spike Prime con Pybricks
- 4-5 runs con cambio de attachment
- Scoring por misiones completadas

### Tips clave (de nuestra investigación)

- Auto-avanzar entre runs ahorra 5-10 segundos
- Consistencia > velocidad
- Alinear a pared antes de cada run
- Timeout por misión para no perder tiempo
- Plan B para cada misión
