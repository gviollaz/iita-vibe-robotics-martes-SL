# 🌍 WRO RoboMission

## Referencia para el taller

### Resumen

- Robot max 250×250×250mm antes de arrancar
- LEGO parts + controladores LEGO
- Enfasis en seguimiento de línea y detección de color
- Reglas sorpresa el día de competencia
- Robot construido desde cero en cuarentena (internacionales)

### Categorías por edad

| Categoría | Edad | Complejidad |
|-----------|:----:|:-----------:|
| Elementary | 8-12 | Baja-Media |
| Junior | 11-15 | Media-Alta |
| Senior | 14-19 | Alta |

### Skills clave

1. Line following PID (esencial)
2. Detección de colores con calibración in-situ
3. Máquinas de estado para decisiones
4. Arrays para almacenar secuencias detectadas
5. Código robusto ante variaciones del campo
